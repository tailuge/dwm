# Luke's fork of Luke's build of dwm

Easy way to setup DWM on raspberry pi os.

```
git clone https://github.com/tailuge/dwm
cd dwm
sudo ./setup-pi.sh
sudo make install
```

## pi useful

```
git config --global user.name tailuge
sudo dpkg-reconfigure sddm
lsusb
lsblk


```
